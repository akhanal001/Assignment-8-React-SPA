# Assignment-8-React-SPA


### How to Run the Project Locally

1. Download or clone this project to your computer.
     ```bash
    https://github.com/akhanal001/Assignment-8-React-SPA.git
    ```
3. Open the project folder in a code editor.
4. Make sure Node.js is installed on your system.
5. In the terminal, run:

    ```bash
    npm install
    ```

6. Then start the development server:

    ```bash
    npm run dev
    ```

7. Open the link shown in the terminal to view the app.
    Link like this :
    ```bash
    http://localhost:5173/ 
    ```
